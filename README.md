# WATResearch Web

The official website for WATResearch, built with React, TypeScript, and Vite. Deployed on Vercel.

**Live:** [watresearch.ca](https://watresearch.ca)

## Tech Stack

- **Frontend:** React-TypeScript + TailwindCSS
- **Build Tool:** Vite
- **Hosting:** Vercel

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

## Scripts

| Command | Description |
| :--- | :--- |
| `npm run dev` | Start dev server |
| `npm run build` | Type-check and build for production |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview production build locally |

## Deployment

Vercel auto-deploys on push to `main`. Preview deployments are created for pull requests.
